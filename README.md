# Senior-Project

My undergraduate senior capstone: a Unity project (C# scripts and GLSL/shader work).

## Layout

- `Assets/`: Unity assets, scenes, scripts, and shaders
- `ProjectSettings/`: Unity project settings
- `Assembly-CSharp*.csproj`: generated solution/project files

## Open it

Open the project folder in the Unity Editor (the version it was built with, circa 2015).
